Principais formatações

Negrito / Destaque

Duas mandeiras de colocar negrito: usando a Tag b(bold) que não é um jeito semantico

E com a opção strong, que ja é de uma forma semântica mostra que dentro do texto, esse termo destacado tem um significado forte

Itálico / Ênfase

Para usar o italico apertar CTRL SHIFT P, escolher Wrap with abbreviationnão semantico.

Já com a tag em(semântica) para deixar em ênfase
